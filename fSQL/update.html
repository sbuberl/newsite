UPDATE Query

Update one or more rows into a table


UPDATE [IGNORE] tableName
SET columnName=columnValue[ ,... ]
[WHERE condition]

Examples:

To update all the rows in the table


UPDATE options SET field_name = 'max_active_peers', field_data = '5'

To update some of the rows in the table


UPDATE options SET field_data = '10' WHERE field_name != 'graph_data_range_sent'

If a primary key violation occurs during an UPDATE, an error is returned.


UPDATE options SET field_name = 'timeout'
UPDATE options SET field_name = 'timeout'

Since the field_name column is the primary key, after the first UPDATE passes, the next one will fail because duplicate values for the primary key. To make this error ignored, use the IGNORE keyword


UPDATE IGNORE options SET field_name = 'timeout'
