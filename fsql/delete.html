Delete one or more rows into a table


DELETE FROM tableName [WHERE condition]

Examples:

To delete all the rows in the table


DELETE FROM customers

To delete some of the rows in the table


DELETE FROM customers WHERE city = 'derry' OR firstName = 'jon'
